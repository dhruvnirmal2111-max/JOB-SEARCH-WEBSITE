## About Kogan.com

Kogan.com (ASX: KGN) is Australia's largest pure-play online retailer and one of Australia's fastest-growing eCommerce companies. The engineering team builds software that powers the retail, logistics, and customer experience platform for millions of Australians.

---

## Role Overview

We're looking for a Data & ML Engineer to develop and maintain data pipelines, support machine learning workflows, and ensure data integrity across Kogan.com's eCommerce data platform. You'll work across the full data stack — from ingestion pipelines through to ML model deployment and BI reporting.

---

## Key Responsibilities

- Build and maintain scalable data pipelines using Airflow, Cloud Functions, PubSub, and BigQuery on GCP.
- Design, implement, and maintain ETL/ELT workflows using dbt (Data Build Tool).
- Support and build ML workflows and model deployment pipelines (SageMaker, Vertex AI).
- Ensure data integrity and data quality across all pipelines and datasets.
- Work with AWS Glue, Databricks, Kafka, and Kinesis for data ingestion and streaming.
- Develop and maintain Looker/Tableau dashboards and reporting datasets for business stakeholders.
- Collaborate with data scientists, analysts, and engineers to support ML feature engineering and model delivery.
- Containerise and deploy data tools using Docker.
- Write clean, maintainable Python code for pipeline and automation work.

---

## Requirements

- Strong Python development skills (3+ years).
- Solid SQL skills for data modelling, querying, and transformation.
- Experience building and maintaining data pipelines (Airflow preferred).
- Exposure to cloud data platforms — GCP (BigQuery, PubSub, Cloud Functions) and/or AWS (Glue, Kinesis, SageMaker, S3).
- Experience with dbt or similar SQL transformation frameworks.
- Understanding of ML pipeline fundamentals: feature stores, model serving, monitoring.
- Experience with Databricks or Spark for large-scale data processing.
- Comfortable with Docker for containerisation.
- Strong data quality mindset.

## Nice to Have

- Experience with eCommerce data at scale.
- Exposure to Kafka or streaming data architectures.
- Experience with Looker or BI tooling.
- Familiarity with Vertex AI or SageMaker for ML model deployment.
- Django or backend Python framework experience.
