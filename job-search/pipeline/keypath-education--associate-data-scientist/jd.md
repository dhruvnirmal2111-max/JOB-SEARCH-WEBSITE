# Keypath Education — Associate Data Scientist (Strategy & Insights)

_Source: pasted JD. Captured 2026-07-16. Keypath Education = global EdTech / OPM (online program management), ASX-listed, Melbourne HQ. Foundational/early-career DS role; reports to a Data Scientist._

## The opportunity

Associate Data Scientist on the Strategy & Insights team. A foundational role for someone at the start of their data science career, embedded in real commercial projects from day one — predictive models, generative AI solutions, and analytical work that shapes how Keypath operates.

Open to two candidate types:
- **Career transitioners:** background in analytics, research, engineering, or a quantitative discipline, completing (or completed) a Master's to formalise the move into data science. Structured thinking, analytical maturity, professional experience alongside postgraduate training.
- **High-potential graduates:** recent/upcoming Master's grads with a strong academic foundation in DS/CS/stats/quant, lighter on professional experience but with the technical ability, curiosity, and ambition to grow into a leader.

## What you'll be doing

- Support predictive models across the student lifecycle: lead scoring, conversion propensity, retention risk, demand forecasting.
- Assist in building + evaluating generative AI solutions: LLM-powered agents, prompt-engineering workflows, retrieval-augmented generation (RAG) pipelines.
- Exploratory data analysis, feature engineering, data wrangling to prep datasets for modelling.
- Write clean, well-documented Python for analysis + model development; contribute to shared codebases and notebooks.
- Help productionise models: pipeline development, testing, integration with downstream systems.
- Prepare visualisations, summaries, presentations that communicate findings + model outputs to non-technical stakeholders.
- Contribute to the knowledge base: document methods, share learnings, participate in code reviews.
- Proactively develop your own skills through structured learning, certifications, hands-on experimentation.

## About you

_"We don't expect you to tick every box. We care far more about how you think, how you learn, and what you're capable of becoming."_

### Essential
- Bachelor's or Master's (completed/near-completion) in DS, statistics, CS, maths, engineering, economics, or a related quantitative field.
- Python for data analysis + modelling (pandas, NumPy, scikit-learn or equivalent) — evidence you can write functional code, not perfection.
- Working understanding of core stats + ML concepts: regression, classification, model evaluation, overfitting, train/test methodology.
- Strong analytical thinking + problem-solving — break an ambiguous question into a structured approach.
- Clear written + verbal communication — explain reasoning + findings to a non-technical audience.
- Genuine curiosity about DS/AI and how models drive real decisions — active engagement via projects, study, writing, or community.

### Desirable
- Exposure to LLMs, prompt engineering, or generative AI tools/concepts.
- SQL for data extraction + manipulation.
- Version control (Git) + collaborative development.
- Cloud, particularly Microsoft Azure or Microsoft Fabric.
- A strong capstone/thesis/applied research project showing end-to-end DS (Kaggle, GitHub repos, published research count).
- Previous professional experience in analytics, research, consulting, or a quantitative role (for career transitioners).
